📊 Google Sheets — Orders & Customers Not Connected

Paste your Google Apps Script URL below to auto-save every checkout and customer record to your spreadsheet. The URL is injected into js/shared.js immediately on save.

Quick Setup
  1. Create a Google Sheet with tabs: Customers and Orders
  2. Open Extensions → Apps Script and paste the script below
  3. Click Deploy → New deployment → Web App (Anyone can access)
  4. Copy the Web App URL and paste it here
function doPost(e){
  const d=JSON.parse(e.postData.contents);
  const ss=SpreadsheetApp.getActiveSpreadsheet();
  const sh=ss.getSheetByName(d.sheet)||ss.insertSheet(d.sheet);
  const keys=Object.keys(d.data);
  if(sh.getLastRow()===0) sh.appendRow(['Timestamp',...keys]);
  sh.appendRow([d.timestamp,...keys.map(k=>d.data[k]||'')]);
  return ContentService.createTextOutput(JSON.stringify({status:'ok'})).setMimeType(ContentService.MimeType.JSON);
}
function doGet(){return ContentService.createTextOutput('PDS Sheets API Active');}

🎟 Promo & Discount Manager

Create and edit promo codes for small orders and bulk discount tiers for large orders. All changes go live instantly.

🏷 Small Orders — Promo Codes

Promo codes are entered by customers at checkout. Each code can give a % discount, a fixed ₦ amount off, or free delivery. Codes are case-insensitive.

📦 Bulk Orders — Automatic Quantity Tiers

Bulk discounts are applied automatically at checkout based on total item quantity — no code needed. Set minimum quantity thresholds and the % off applied at each level.

ℹ How it works A customer buying 25 units hits the 25+ tier automatically. Tiers stack from lowest to highest — the highest matching tier wins.
